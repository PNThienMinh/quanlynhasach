Hướng dẫn cấu hình phần mềm:
B1: Cài đặt phần mềm bằng file setup
B2: Vào thư mục lưu phần mềm, tìm đến thư mục Database, chạy đoạn script sql bằng SQL Server (tối thiểu 2016)
B3: Mở phần mềm, ở lần sử dụng đầu tiên, bạn cần cấu hình server của cơ sở dữ liệu. Lấy thông tin cấu hình ở B4
B4: Lấy thông tin server của cơ sở dữ liệu (ví dụ: DESKTOP-0HNVS4D\SQL2016), copy và paste vào trong cửa sổ hiện ra. Bấm "Test", nếu có thông báo thành công thì bấm OK để khởi động lại phần mềm
B5: Enjoy!
